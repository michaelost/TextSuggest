Install

Ubuntu and Debian

Get the deb file: Download textsuggest-git.deb

Arch Linux

Get it from the AUR: textsuggest-git

Manual

Get the code and run sudo ./install.sh.


Using TextSuggest

Simply bind the command textsuggest --auto-selection to a keyboard shortcut. Type a word, press the shortcut. See TextSuggest give you suggestions.


Advanced usage

TextSuggest is flexible and has many options. See the README for full documentation.
